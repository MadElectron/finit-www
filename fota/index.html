FIN-IT FOTA

Мощный инструмент дистрибуции приложений на устройства Adnroid

Тихая
установка

Приложение устанавливаются на устройство пользователя автоматически, иконка приложения размещается на первом рабочем столе в случае свободного места на нем

Установка с уведомлением

Приложение устанавливаются на устройство автоматически, иконка приложения размещается на первом рабочем столе в случае свободного места на нем. Вместе с установкой выводится push-сообщение о том, что новое приложение добавлено. Один или несколько раз.

Обновление
прошивки

Пользователь получает уведомление о появлении новой версии прошивки, по клику происходит обновление устройства. Обновление прошивки используется для исправления багов и/или для установки на устройство нужных системных приложений.
